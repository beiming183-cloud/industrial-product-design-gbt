# Concept decision — ORBIT cable dock

All three massings were built as real B-rep STEP models and reviewed at a common isometric convention. The user delegated selection for this public case study.

| Criterion | A — static comb | B — tall drum | C — low orbit |
| --- | ---: | ---: | ---: |
| Four open cable captures | PASS | PASS | PASS |
| Rotary design DNA | FAIL | PASS | PASS |
| Footprint below 100 × 100 mm | FAIL (108 mm) | PASS | PASS |
| Height below 30 mm | PASS | FAIL (53.8 mm) | PASS (26 mm massing) |
| Low visual centre | PASS | FAIL | PASS |
| Motion reads without explanation | FAIL | PASS | PASS |
| Service/assembly path | simple | moderate | moderate |
| Key risk | no rotary benefit | top-heavy and tower-like | detent and cable fit need prototype evidence |

Decision: select Concept C. It is the only direction that preserves the rotary interaction, fits the target footprint, and remains visibly low and stable. A is retained as the simplicity benchmark. B is retained as evidence that compact footprint alone does not make a convincing desktop product.

The selected C massing authorized detailed CAD because the design authority was explicitly delegated. External user appearance approval is still pending and is not relabeled as complete.
